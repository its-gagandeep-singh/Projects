# ❌ Tic-Tac-Toe Game ⭕

A clean, responsive, and interactive classic Tic-Tac-Toe game built from scratch using frontend web technologies. This project was developed as a mini-project to reinforce core development fundamentals.

## 🚀 Live Demo

Check out the live application here: **[Live Demo Link](https://lnkd.in/g2M2WfuK)**

---

## 🛠️ Features

- **Interactive Gameplay:** Fully functional grid system allowing two players to play locally.
- **Responsive Layout:** Designed using CSS Flexbox to ensure a seamless experience across different screen sizes.
- **Win/Draw Detection:** Smart JavaScript logic that instantly evaluates win conditions or handles draw scenarios.

---

## 💻 Tech Stack

- **HTML5:** Used to build the core semantic structure of the game interface.
- **CSS3:** Leveraged for layout positioning (Flexbox), modern styling, and clean visual presentation.
- **JavaScript (ES6):** Used to handle user interactions, update the game state, and execute the core game logic.
