## 🚀 Projects List

### 1. Tic Tac Toe Game
A classic, interactive, responsive Tic Tac Toe game built using core web technologies.
*   **Technologies Used:** HTML5, CSS3, JavaScript (ES6+)
*   **Key Features:** 
    *   Clean, modern, minimalist UI with dynamic background styling.
    *   Responsive design optimized for both mobile and desktop screens.
    *   Interactive grid layout using responsive viewport sizing (`vmin`).
